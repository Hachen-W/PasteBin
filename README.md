# PasteBin

Локальная система для хранения блоков текста и удобного шаринга этих блоков по ссылке

Объём текста до 10 Мб

**Требования к системе:**
1. Возвращаемые хэши в URL к постам должны быть уникальные и короткие
2. Некоторые посты будут гораздо популярнее других
3. Одни пользователи будут создавать посты чаще других

Запуск программы:
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
