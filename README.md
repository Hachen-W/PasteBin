# PasteBin

Локальная система для хранения блоков текста и удобного шаринга этих блоков по ссылке

Объём текста до 10 Мб

**Требования к системе:**
1. Возвращаемые хэши в URL к постам должны быть уникальные и короткие
2. Некоторые посты будут гораздо популярнее других
3. Одни пользователи будут создавать посты чаще других

**Запуск программы:**
```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
docker compose up -d
uvicorn main:app --reload
```

## Устройство

**Архитектура:**
- FastAPI выступает в роли диспетчера, принимая и отдавая данные.
- PostgreSQL хранит метаданные: уникальные ID и пути к файлам.
- Локальная файловая система (папка pasted_texts) для хранения тяжелых текстовых блоков.
- Secrets создает ссылки для шаринга.
- Модуль периодической очистки устаревших данных
- Шифрование пасты при получении и дешифрование при отправке
